# 贡献者行为准则

## 我们的承诺

我们致力于为每个人提供一个友好、安全和包容的环境。

## 我们的标准

**积极行为包括：**
- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情

**不可接受的行为包括：**
- 使用性暗示的语言或图像
- 人身攻击、侮辱或贬低性评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息

## 执行

如发现不当行为，请通过 [GitHub Issues](../../issues) 联系项目维护者。

本行为准则改编自 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
