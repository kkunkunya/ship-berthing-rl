### 验收标准（演示版）

| 指标 | 无干扰基线 | 有干扰演示目标 | 备注 |
|------|-----------|---------------|------|
| 位置误差 | < 0.1 m | **< 0.8 m** | 放宽至合理范围 |
| 航向误差 | < 2° | **< 10°** | 允许补偿姿态 |
| 最终速度 | < 0.015 m/s | < 0.05 m/s | 略放宽 |
| 偏航角速度 | < 0.015 rad/s | < 0.03 rad/s | 略放宽 |
| **成功率** | ~100% | **≥ 50%** | 固定扰动下可复现 |
| 奖励收敛 | -30000 → 0 | 稳定收敛趋势 | 允许波动 |

**干扰配置**：
- 海流：Vc = 0.075 m/s（**固定方向**）
- 风力：Vw = 1.2 m/s（**固定方向**）
- 模式：`current_mode=fixed`, `wind_mode=fixed`

### 确定的技术方案（渐进式爬坡 + 固定扰动）

**核心思路**：使用内置 `ramp_disturbance` 从弱到强逐级训练，最终固定在演示配置。

#### 训练阶段
1. **从 Stage 2 继续训练**（基线：Vc=0.05, Vw=0.8, 成功率90%）
2. **渐进式爬坡**（3小时限制）
   - 海流梯度：0.05 → 0.06 → 0.07 → 0.075 m/s
   - 风力梯度：0.8 → 1.0 → 1.15 → 1.2 m/s
   - 成功率阈值：每级达到 60% 才上调
3. **固定扰动微调**（最后阶段）
   - 切换为 fixed 模式
   - 固定 Vc=0.075, Vw=1.2
   - 提升可复现性

#### 允许调整项（用户确认）
- 奖励函数（包括终端奖励、零转速奖励、权重比例）
- 课程容差（位置/航向/速度/偏航角速度阈值）
- 训练策略（爬坡阈值/窗口/耐心、固定→随机切换条件）

### 交付内容
- 训练得到的模型 checkpoint（含最终模型与最优评估模型）
- 评估结果（成功率与终端指标统计，方向覆盖评估）
- 项目整理与索引文档（模块结构、调用关系、关键文件说明）
- 可复现实验命令清单
- 所有运行命令输出日志（`.txt`，存放在 `log/`）

## 📎 附录：日志与命令规范

### 日志规范
- 所有训练/评估命令输出必须写入 `log/*.txt`
- 实时监控使用 `Get-Content -Wait`

### 命令模板（示例）
```powershell
# 训练（输出写入 log）
$ts = Get-Date -Format "yyyyMMdd_HHmmss"
$log = "log/train_$ts.txt"
uv run python -m src.train.train_vectorized ... 2>&1 | Tee-Object -FilePath $log

# 评估（输出写入 log）
$elog = "log/eval_$ts.txt"
uv run python scripts/analyze_failure_modes.py ... 2>&1 | Tee-Object -FilePath $elog

# 实时查看日志
Get-Content -Wait log\train_YYYYMMDD_HHMMSS.txt
```
